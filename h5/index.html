# input 类型标签可以做表单验证：

  1. text：文本输入框，用于输入单行文本。

  2. password：密码输入框，用于输入密码，输入内容会以星号或圆点的形式显示。

  3. number：数字输入框，用于输入数值。

  4. email：电子邮件输入框，用于输入电子邮件地址。

  5. tel：电话号码输入框，用于输入电话号码。

  6. date：日期输入框，用于选择日期。

  7. time：时间输入框，用于选择时间。

  8. radio：单选按钮，用于选择一项。

  9. checkbox：复选框，用于选择多项。

  10. file：文件上传输入框，用于选择上传文件。

  11. submit：提交按钮，用于提交表单数据。

  12. reset：重置按钮，用于重置表单数据。

  13. button：普通按钮，用于触发自定义的 JavaScript 代码。

  14. hidden：隐藏输入框，用于存储不显示给用户的数据。

  15. color：颜色选择器，用于选择颜色。

  16. range：滑动条，用于选择一个范围内的值。

  17. search：搜索输入框，用于输入搜索关键字。

  18. url：URL 输入框，用于输入 URL 地址。

除了以上常用的 type 属性值，还可以使用一些 HTML5 新增的 input 类型，如 datetime、datetime-local、month、week 等。
