# 3.GIT 分支（重点）第三次记录

分支
是指向提交节点的可变指针，默认名字是 Master

我们使用 git log --oneline 的时候也看到有（HEAD->master）指向的是最后一次的提交记录，

Head 指针会影响工作区和暂存区的代码状态

作用场景：
开发新需求或者修复 bug, 保证主线代码随时可以使用，多人协同提高开发效率
